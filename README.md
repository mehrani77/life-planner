# Life OS Planner

یک PWA سبک و لوکال فرست برای برنامه ریزی روزانه، ماهانه و سالانه.

نسخه فعلی با فضای Daily Planner سرمه ای/طلایی طراحی شده و شامل اولویت های امروز، Schedule بیست و چهار ساعته، Focus Tasks، هدف های قابل نوشتن، یادگیری، چالش ها، Habit Tracker و مرور روز است.

تسک ها فیلد ساعت دقیق و مدت زمان دارند. اگر برای تسک ساعت بنویسید، همان تسک به صورت خودکار در جدول Schedule همان روز و همان بازه ساعتی نمایش داده می شود. تسک های امروز که هنوز ساعت ندارند در بخش Unscheduled دیده می شوند.

صفحه روزانه فقط تسک های همان تاریخ انتخاب شده را نشان می دهد. با دکمه های «روز قبل»، «امروز» و «روز بعد» می توانید کارهای روزهای قبلی یا بعدی را ببینید و ویرایش کنید. تقویم هم دکمه ماه قبل/بعد و دکمه «باز کردن روز» دارد.

بخش ماهانه از اطلاعات تسک های روزانه همان ماه نمودار دایره ای و گزارش هفتگی می سازد. بخش سالانه هم همان داده ها را در سطح سال جمع بندی می کند و چند شاخص سالانه قابل ویرایش دارد.

## استفاده روی آیفون

1. پوشه را روی GitHub Pages یا هر هاست استاتیک قرار دهید.
2. لینک را با Safari باز کنید.
3. از دکمه Share گزینه Add to Home Screen را انتخاب کنید.
4. اپ از صفحه اصلی آیفون اجرا می شود و داده ها را روی همان دستگاه ذخیره می کند.

## انتشار روی GitHub Pages

1. یک repository جدید بسازید.
2. فایل های این پوشه را در ریشه repository قرار دهید.
3. در GitHub به Settings > Pages بروید.
4. Source را روی Deploy from a branch بگذارید.
5. Branch را روی main و folder را روی /root تنظیم کنید.

## داده ها

داده ها در مرورگر با localStorage ذخیره می شوند. برای بکاپ از دکمه خروجی داخل اپ استفاده کنید و فایل JSON را نگه دارید. برای بازیابی، دکمه ورودی را بزنید.

## تسک صوتی

در صفحه داشبورد یا امروز، دکمه «تسک صوتی» را بزنید و جمله ای مثل این بگویید:

`عنوان تماس با علی برای فردا اولویت بالا حوزه کار توضیح پیگیری قرارداد`

برای ثبت ساعت و مدت هم می توانید بگویید:

`عنوان تماس با علی برای فردا ساعت ۱۴:۳۰ مدت ۳۰ دقیقه اولویت بالا حوزه کار توضیح پیگیری قرارداد`

اپ عنوان، تاریخ، ساعت، مدت، اولویت، حوزه، وضعیت و یادداشت را تا جای ممکن خودش پر می کند. برای باز شدن میکروفون روی گوشی، بهتر است اپ از مسیر HTTPS مثل GitHub Pages اجرا شود. اگر مرورگر تشخیص صدای مستقیم را پشتیبانی نکند، داخل کادر متن بزنید و از میکروفون کیبورد آیفون برای دیکته استفاده کنید.
